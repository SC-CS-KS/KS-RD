* 阿姆达尔定律 (Amdahl's Law)   

阿姆达尔定律是一个显示计算任务潜在加速能力的公式。   
这种能力可以通过增加系统资源来实现，通常用于并行计算中。  
它可以预测增加处理器数量的实际好处，然而增加处理器数量会受到程序并行性的限制。
